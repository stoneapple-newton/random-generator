# random-generator
this repository is a C++ light-weight random library.

# How to use
1.install this `ran_gen.h` file.
2.write `ran_gen_c ran_gen;`.
3.initalize `ran_gen`.`ran_gen.srand();`
4.use it.e.g.
`ran_gen.rand16()`
`ran_gen.rand32()`
`ran_gen.rand64()`
`ran_gen.randf32()`
`ran_gen.randf64()`
